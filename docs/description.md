# Overview

Performs rarefaction analysis on immune receptor repertoires (TCR or BCR) or peptide data to assess library diversity and compare richness across samples with different sequencing depths. Rarefaction is a statistical technique that estimates the expected number of unique sequences as a function of sampling depth, enabling fair comparison of diversity between libraries of different sizes.

The block processes count data from upstream VDJ or peptide profiling blocks and performs random subsampling at multiple depths for each sample. At each subsampling depth, the number of unique sequences is recorded, and this process is repeated multiple times to generate smooth rarefaction curves representing the mean expected number of unique sequences. The shape and slope of these curves provide insights into library diversity: steeper curves indicate higher diversity and suggest that additional sequencing would reveal more unique sequences, while plateauing curves indicate that most diversity has been captured at the current sequencing depth.
